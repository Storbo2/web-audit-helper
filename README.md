# WAH – Web Audit Helper 🧠 (WIP)

WAH (Web Audit Helper) is a framework-agnostic JavaScript/TypeScript library designed to help developers audit web pages during development.

It focuses on:
- Accessibility
- Responsive design
- Semantic HTML
- Basic SEO best practices

WAH provides a configurable visual overlay, DOM indicators, console logs and exportable reports, making it ideal for real-world frontend development.

---

## ✨ Features

- ✔️ Accessibility checks (contrast, font size, labels, alt, headings)
- 📱 Responsive helpers with customizable breakpoints
- 🧱 Semantic HTML analysis
- 🔍 Basic SEO checks
- 🎨 Floating overlay with visual indicators
- ⚙️ Fully configurable (colors, positions, rules)
- 📊 Scoring system
- 📤 Export reports (JSON / TXT)
- 🧩 Framework-agnostic
- 🧹 Zero runtime dependencies
